#  Cryptocurrency Market Data Analysis with Python

##  Project Overview

This project demonstrates how to retrieve real-time cryptocurrency market data from the CoinMarketCap API using Python and perform data cleaning, transformation, and exploratory data analysis (EDA) with Pandas.

The goal of this project is to showcase practical data analysis skills, including working with APIs, processing JSON data, automating data collection, and preparing datasets for analysis.

##  Project Workflow

1. Connected to the CoinMarketCap API using Python.
2. Retrieved live cryptocurrency market data.
3. Converted nested JSON responses into a structured Pandas DataFrame.
4. Added timestamps to track data collection.
5. Exported the data to CSV.
6. Automated repeated API calls to collect historical market data.
7. Performed data transformation and aggregation using Pandas.
8. Prepared the dataset for reporting and visualization.
